Q:1 How javascript works?

javascript is a high level programming language which makes web site interactive.Whenever we run a JavaScript program inside a web browser, the source code goes through several steps and gets executed by js engine.

step 1:Parser

It is the first stage of the engine. The parser's job is to check the JavaScript code for syntactic errors in line by line manner because JavaScript is an interpretive scripting language, so whenever an error is detected by the parser, it throws a kind of error and stops execution of the code.

step 2:AST

After parsing it creates the data structure called AST (Abstract Syntax Tree).

Step 3: Conversion to Machine code

Once the Abstract Syntax Tree is created by the parser, the JavaScript engine converts the JavaScript code into the machine code.

step 4:Execution

after converting into machine language, the converted code is sent to the system for execution, then the byte code run by the system/engine.
